## 友情提醒 🤣
此仓库只是拉取源码进行编译，本人并不会修复相关问题！
<br>源码更新，我也会点一下action.</br>

## 食用教程 🛠

### 1、Fork本仓库 📌
右上角 `Fork`  `star`可以给予鼓励。

### 2、仓库读写权限 🤔

前往Fork之后的仓库页面，按下图实例图示点击，更改读写权限。 

![](doc/img/example7.png)

### 3、开始编译 🚀

前往Fork之后的仓库页面，按下图实例图示点击，等待编译结果即可。

![](doc/img/example4.png)
